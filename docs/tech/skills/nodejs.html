            # 三、安装

            # 1.安装但不写入package.json

            npm install xxx
            1

            # 2.安装并写入package.json的"dependencies"中

            npm install xxx –S
            1

            # 3.安装并写入package.json的"devDependencies"中

            npm install xxx –D
            1

            # 四、全局安装

            npm install xxx -g
            1

            # 五、安装指定版本

            npm install xxx@1.2.0
            1

            # 六、更新

            # 1.先检查更新

            npm outdated
            1

            执行以上命令，可以看到所有可以更新的模块。

            我们需要先更新 package.json文件：

            # 2.安装"npm-check-updates"模块

            npm install -g npm-check-updates
            1

            # 3.检查可更新的模块

            ncu
npm-check-updates
            1
            2

            以上两条命令都可检查可更新模块。接下来更新package.json的依赖包到最新版本：

            ncu -u
            1

            以上命令执行，更新全部模块。但在实际开发中不建议一次全部更新，可以根据实际需要，更新指定的模块，并且可以根据作用范围在后面加上 -D、-S 或 -g

            npm update xxx
            1

            注意：指定更新需要提前修改 package.json 中的版本号。

            为了保险起见，package.json 更新后，可删除整个node_modules目录并重新初始化项目。

            npm install
            1

            # 七、删除

            # 删除指定模块

            npm uninstall xxx
            1

            # 删除全局模块

            npm uninstall -g xxx
            1

            # 八、查看

            # 查看全局安装的包

            npm list -g --depth 0
            1

            # 九、启动端口

            http-server -p 8081
            1
