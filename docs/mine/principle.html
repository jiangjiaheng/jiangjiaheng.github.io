        # 原则与行动

        这个页面整合原来的“原则”“三心”“三思而后行”“什么也不做”“新学习方法”“好消息与坏消息”等短笔记。

        它的目标不是建立一套完美理论，而是在日常生活中提供几个能反复使用的提醒。

        # 核心原则

        # 1. 保守主义，攻势足球

        底层要保守，行动要进攻。

        保守主义不是退缩，而是承认现实风险、承认自身限制、承认系统会出错。攻势足球不是鲁莽，而是在明确边界之后主动推进。

        所以更合理的顺序是：

        1. 先守住基本盘
        2. 再寻找进攻机会
        3. 不用一次冒险决定全部命运

        基本盘包括：

        • 个人状态
        • 事业能力
        • 家庭关系
        • 身体健康
        • 金钱安全

        # 2. 勇气，而非暴力

        勇气是面对现实，暴力是被情绪控制。

        真正需要的是：

        • 敢于承认问题
        • 敢于做困难但必要的选择
        • 敢于承担后果
        • 敢于停止错误动作

        不是用激烈情绪证明自己有力量。

        # 3. 不战而胜，先胜后战

        《孙子兵法》里最重要的启发是：不要迷恋战斗本身。

        最好的结果不是拼命赢，而是在行动之前已经创造出更高胜率。

        现实中的对应原则是：

        • 不进入明显劣势的局
        • 不用意志力硬打消耗战
        • 先准备，再行动
        • 先判断胜率，再投入资源

        # 三心

        # 初心

        初心不是口号，而是不要忘记原始目标。

        人很容易被愤怒、面子、比较、短期诱惑带偏，最后追求枝节，忘掉本质。

        提醒自己：

        • 我最初想解决的是什么问题
        • 现在的动作有没有偏离目标
        • 我是在推进事情，还是在证明情绪

        # 耐心

        真正重要的工作往往只有两件：

        1. 准备
        2. 等待

        很多失败不是能力不足，而是太急、太想立刻得到结果、太早做了不该做的动作。

        耐心不是被动，而是允许事情在正确时间发生。

        # 贪心

        贪是人性的大弱点。

        贪心常见的表现是：

        • 急于求成
        • 想要更多
        • 不自量力
        • 不切实际
        • 不愿意接受正常速度

        减少贪心，本质上是在减少系统风险。

        # 决策方法

        # 三思而后行

        冲动、矛盾和重大决定出现时，先问三个问题：

        1. 是不是必须这么做
        2. 这么做之后有什么后果
        3. 有没有替代方案

        当内在冲突出现，大脑和身体其实已经在提醒自己放慢速度。

        三思而后行不是拖延，而是给理性重新接管系统的时间。

        # 没有消息就是好消息

        生活平淡、没有强烈变化时，不一定是坏事。

        很多突然出现的消息，往往意味着某种风险、麻烦或外部变量已经发生。

        所以在平淡和寂寞时，可以提醒自己：

        没有消息，往往就是系统仍然稳定。

        # 什么也不做

        什么也不做，有时也是一种解决方式。

        适用场景：

        • 钻牛角尖时
        • 无能为力时
        • 太累时
        • 情绪已经过载时
        • 信息不足时

        不行动不是放弃，而是避免在低质量状态下制造更多问题。

        # 学习方法

        学习分三步：

        1. 总结、提炼
        2. 反思、内化
        3. 运用、实践

        总结提炼，是把书和知识读薄。

        反思内化，是把别人的语言变成自己能理解的语言。

        运用实践，是让知识进入现实，而不是停留在笔记里。

        适用范围：

        • 读书
        • 学英语
        • 学技术
        • 学任何长期技能

        # 时间感

        过去、现在和未来不是完全分离的。

        • 过去影响现在
        • 现在决定未来
        • 未来会重新理解过去

        所以最重要的动作不是沉迷过去，也不是幻想未来，而是把现在过好。

        Enjoy the moment.

        # 一句话总结

        行动原则的核心不是让自己更激烈，而是让自己在复杂现实里更稳、更慢、更准。
