        # 新学习方法

        # 含义

        1. 总结、提炼
        2. 反思、内化
        3. 运用、实践

        # 解释

        1. 通过总结提炼把书或者知识读薄
        2. 反思内化把知识融入自己的内心，用自己能理解的方式学会
        3. 最后在现实生活中实践运用

        # 用途

        1. 读书
        2. 学习英语
        3. 任何知识
