# 自控力

# 书中经典汇总

# 1. 关于意志力

  1. 自知之明是指控的基础。认识到自己的意志力存在问题，则是自控的关键。

  2. 意志力本能：三思而后行。

  3. 你意识到内在冲突的时候，大脑和身体会做出反应，帮助你放慢速度、抑制冲动。

  4. 对跑步者而言，锻炼对意志力的效果是立竿见影。15分钟的跑步机锻炼就能降低巧克力对节食者、香烟对戒烟者的诱惑。锻炼的长期效果更加显著。

  5. 研究人员发现，人们早晨的意志力最强，随后意志力随着时间的推移逐渐减弱。

  6. 资源不足时，大脑会选择满足当下的需求；资源充足时，大脑则会转向选择长期的投资。

  7. 当你的血糖含量降低时，你的大脑仍旧会考虑短期的感受，会去冲动行事。

  8. 每个工作日都吃一顿丰盛健康的早餐，而不是什么都不吃；吃零食时选择坚果，而不是选择糖果，你获得的意志力都会比你消耗的多。

  9. 每天都用力握一个把手、戒掉甜食、记录支出情况。

  10. 关注自己正在做的事情、选择更难而不是最简单的事。

# 2. 关于压力

  1. 从长远的角度看，没有什么比压力更加消耗意志力了。
  2. 压力让你关注即时的、短期的目标和结果，自控力则需要你的大脑有更广阔的视野。
  3. 学会如何更好地管理压力，是提高意志力的重要组成部分。
  4. 缓解压力最常见的方式就是那些能够激活大脑奖励系统的方式——吃东西、喝酒、购物、看电视、上网和 玩游戏。
  5. 现实世界的压力会让戒烟、戒酒、节食的人更容易重蹈覆辙。
  6. 压力包括愤怒、悲伤、自我怀疑、焦虑等消极情绪。
  7. 压力和多巴胺的强强联手，让我们一次又一次地败下阵来。
  8. 缓解压力最有效的方式：锻炼、祈祷、阅读、听音乐、与家人朋友相处、按摩、外出散步、冥想货做瑜伽、以及培养有创意的爱好。
  9. 真正能缓解压力的不是释放多巴胺、而是增加大脑中改善情绪的化学物质，如血清素、Y-氨基丁酸和让人感觉良好的催产素。
  10. 在你感到压力前，多鼓励自己。
  11. 我们需要允许自己去做真正让自己快乐的事，远离那些与我们生活无关的压力根源。

# 3. 关于疲惫

  1. 运动疲惫的原因或许不是肌肉无法继续工作，而是大脑中过度保护性的监控机制发挥了作用。
  2. 疲惫不是一种身体反应，而是一种感觉、一种情绪。
  3. 疲惫时一种预先警报系统，所以极限运动员能不断突破常人眼中的身体极限。
  4. 当你的意志力告急时，你可以挖掘你的“我想要”的力量，让自己恢复能力。

# 4. 关于诱惑、道德

  1. 有时，我们是有意识地选择了在诱惑面前屈服。
  2. 乐观精神往往会允许我们放纵自己，觉得自己品德高尚反而是通往罪恶的快速通道。
  3. 我们只想让自己感觉良好，而这就是为自己的胡作非为开了绿灯。
  4. “道德许可”不仅会批准我们做坏事，也会让我们错失做善事的机会。
  5. 大部分人在觉得自己品德高尚时，都不会质疑自己的冲动。而一些人的工作总能让他们觉得自己品德高尚。
  6. 已经这么好了，应该得到一点奖励。这种对补偿的渴望常常使我们堕落。
  7. 实际上，我们相信这种感觉：我一直是善良的，一直是个好人。
  8. 如果你自控的唯一动力就是成为一个足够好的人，那么每当你感觉良好的时候，你就会放弃自控。
  9. “道德许可”最糟糕的部分并不是它可疑的逻辑，而是它会诱使我们做出背离自己最大利益的事。
  10. 如果你告诉自己，锻炼，存钱或戒烟是件正确的事，而不是件能让你达成目标的事，你就不太可能持之以恒。
  11. 我们用大吃甜食、熬夜晚睡、信用卡负债来判断自己是善还是恶。但是，这些是无法真正体现什么事罪恶，什么是美德。
  12. 当你知道什么会引起欲望的时候，将它放到视线之外，它就不会再吸引你了。
  13. 看着别人在买节日礼物时大手大脚，你的欲望就会增加。
  14. 我们做的每个选择对别人来说也是一种鼓舞或诱惑。

# 5. 关于进步

  1. 心理学家知道，我们总是把进步当成放松的借口。
  2. 进步可能让我们放弃曾经为之奋斗的东西。这是因为，两个自我相互竞争会进一步打破两者之间的平衡。
  3. 实际生活中，前进一小步会导致你退后两大步。
  4. 这不是说进步本身是个问题，问题在于进步给我们带来的感觉。
  5. 你要清楚自己做了什么，并紧盯自己的目标。
  6. 这种习惯允许我们今天犯错，明天补救——我们就是这么告诉自己的。

# 6. 关于多巴胺

  1. 当多巴胺劫持了你的注意力时，大脑只会想如何获得或者重复那个触发它的东西。
  2. 大脑正是靠对快乐的承诺让你不停地去狩猎、采集野果、工作和求爱。
  3. 每当我们看到、闻到货尝到高糖分的食物时，大脑都会释放大量的多巴胺。
  4. 当代科技“及时行乐”的特点，加上原始的激励系统，就让我们成了多巴胺的奴隶。

# 7. 关于欲望

  1. 情色图片会使男性更容易在经济方面冒险。
  2. 虽然我们所处的世界总能让我们产生欲望，但是我们只要用心观察，就能看透一些东西。
  3. 欲望时大脑的行动战略。正如我们看到的，它可能对自控构成威胁，也可能是意志力的来源。
  4. 欲望没有绝对的好坏之分，重要的是欲望将我们引向哪个方向，以及 我们是否足够明智，知道什么时候该听从欲望的声音。
  5. 无论你想摆脱哪种恐惧或欲望，它们最后都会变得更让人信服，更引人注目。

# 8. 远离社交媒体

  1. 请花上24消失，远离那些让你产生恐惧的电视新闻、访谈节目、杂志或者网页。

# 9. 自我原谅

  1. 导致更多堕落行为并不是第一次的放弃，而是第一次放弃后产生的羞耻感、罪恶感、失控感和绝望感。
  2. 自我谅解能帮助人们从错误中恢复过来，因为它能够消除热门想到失败时的羞愧和痛苦。
  3. 你只是一个凡人。每个人都会遇到意志力挑战，每个人都有失去自控的时候。这只是人性的组成部分，挫折并不是意味着你本身有问题。
  4. 花点时间，仔细想一想你改变自己的动力和期望。你只有在情绪低落时才会有动力改变吗。

# 10. 关于自己

  1. 如果我们想作出更加明智的决定，就要更好地理解和支持未来的自己。
  2. 认识自我、关心自我和提醒自己真正重要的事物，这三种方法正是自我控制的基石。
  3. 如果我们想要获得心灵的平静和足够的自控力，我们就需要认识到，控制自己的思想是件不可能的事。我们能做的就是，选择自己相信什么，选择自己要做什么。
  4. 自控力的关键就是理解这些不同的自我，而不是从根本上改变自己。
  5. 自控力最强的人不是从与自我的较量中获得自控，而是学会了如何接受相互冲突的自我，并将这些自我融为一体。
