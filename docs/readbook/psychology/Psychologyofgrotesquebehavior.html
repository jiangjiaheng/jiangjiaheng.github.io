# 怪诞行为心理学

# 书中经典汇总

# 1. 关于得失

  1. 这反应我们对损失比获得更敏感的心里倾向。
  2. 于是人们对损失要比对相同数量的收益敏感得多，即使股票账户有涨有跌，人们也会频繁地为每日的损失而痛苦，最终将股票抛掉。
  3. 等额的损失造成的痛苦，和等额的收益带来的喜悦相比，损失对人的心理刺激甚至多一倍。
  4. 相对于获得，人们非常不乐意放弃已经属于他们的东西。
  5. 人们可能更多的是受到放弃自己的东西而产生的痛苦的影响。
  6. 不是贪婪，而是嫉妒推动着世界前进。
  7. 承认失败，方可终止失败。

# 2. 关于心理

  1. 损失厌恶：多数人对损失比对收益更敏感。
  2. 确定效应：处于收益状态时，多数人时风险厌恶者。
  3. 反射效应：处于损失状态时，多数人时风险喜好者。
  4. 参照依赖：多数人对得失的判断往往由参照点决定。
  5. 在涉及收益时，我们是风险厌恶者，但涉及损失时，我们却是风险喜好者。
  6. 面对小概率的盈利，多数人时风险喜好者。面对小概率的损失，多数人时风险厌恶者。
  7. 面对确定的获得，人类时风险厌恶者。面对确定的损失，人类是风险追逐者。
  8. 金钱不等式：失去的100元>得到的100元，挣到的100元>白得的100元，今天的100元>明天的100元。
  9. 人们所感受到的“付钱的痛苦”的程度决定了他们是“吝啬鬼”还是”挥霍狂“。
  10. 大数法则是保险公司、赌场、撞骗的骗徒耐以生存的基础。
  11. 大数法则是一种科学；小数法则是一种迷信。
  12. 过度自信导致交易频繁。
  13. 总体而言，吝啬型的人对价格更加敏感，而挥霍型的人则对产品本身的品质以及”购物所带来的乐趣“更加在意。

# 3. 关于幸福

  1. 我们快乐和不快乐的根源，都是和别人去攀比。向下攀比是幸福，向上攀比是痛苦的。
  2. 别无选择与选择过多皆是烦恼。
  3. 面对的选择越多，就越难取舍。
  4. 人在上了年纪以后，事业成功带来的幸福感会越来越小，而感情寄托、家庭价值日益显得重要。
  5. 如果非要每一步都做得绝对理性，每一步都不踏错，就注定茫然、崩溃。
  6. 选择你爱的，你将会更爱你所选择的。

# 4. 关于金钱

  1. 要有100%的利润，庸众才会蠢蠢欲动。
  2. 越有钱，越是一毫不肯放松，越是一毫不肯放松，便越有钱。
  3. 因贪婪而投资，因亏损而惜售，因希望而等待，因小利而放弃。如此往复，钱越变越少，不变的只有行为的方式。

# 5. 关于谈判

  1. 谈判等方面，低标准的目标往往使人谨慎行事，高标准的目标往往使人敢于冒险。
  2. 低买高卖，是最基本的商业法则。但涉及大宗投资的时候，一个人的决策就不见得那么理性。
  3. 定一个荒唐的天价，再给一个非常优惠的折扣，就能大大促进商品的销量。虽然是简单至极的手段，却是零售业的金科玉律。
  4. 多数人认为，在谈判中最好让对方先开价，这样你就可以去估计对方的低价，可以拥有更多的信息。但事实上，让对方先开价，这个价格就会变成谈判中的一个锚，即使你努力调整，也很难摆脱这个定位效应的影响。
  5. 钓鱼的耐心，政治家的脸皮。
  6. 漫天要价，就地还钱。买卖双方都在有意无意地运用锚定效应，试探，妥协。
  7. 买家在还价时往往锚定在标出的价格之上。如果标出的价格较高，买家在还价时就一般会给出较高的价格，并因砍去了虚头而颇有成就感。

# 6. 二八法则

  1. 二八法则还可以进一步产生作用——4%的人，掌握着全社会64%的财富。
  2. 在真实的社会，运行的是”马太效应“，一凡有的，还要加给他，叫他有余；凡没有的，连他所有的，也要夺取。
  3. 其实，老板应该明白，80%的利润，来自20%的菜品类别，其余的菜品只是陪衬而已。这是管理学中的”二八法则“。

# 7. 锚定效应

  1. 喜出望外，当好事超出预期，人类的大脑就会勃然兴奋。超越预期，其实也是一种锚定效应。
  2. 世界上最动听的一句话，不是我爱你，而是你的癌症时误诊的。
  3. 多数人对得失的判断往往根据参照点决定。
