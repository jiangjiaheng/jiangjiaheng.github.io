# 富爸爸穷爸爸

# 书中经典汇总

# 1. 关于思想

  1. 我注意到穷爸爸之所以穷不在于他挣钱的多少（尽管这也很重要），而在于他的想法和行为。
  2. 造成贫困和财务问题的主要原因是恐惧和无知，而不是经济环境、政府或者富人。人们自身的恐惧和无知使他们困在陷阱里。
  3. 你有两种感情——恐惧和欲望，如果你让它们来控制你的思想，你就会落入一生中最大的陷阱。
  4. 学会让感情跟随你的思想，而不要让思想跟随你的感情。
  5. 所谓的荆棘之路就是指我们的恐惧和贪婪。走进我们的恐惧，直面我们的贪婪、弱点和缺陷是唯一的出路。这条路需要你用心去确定你的思想。
  6. 如果你发现你已在深渊，那你自己就别再挖了。
  7. 记住黄金规则：有黄金的人制定规则。
  8. 大多数人的财务困境是由于随大溜、盲目地跟从其他人所造成的。因此我们都需要不时地照照镜子，相信我们内心的智慧而不是恐惧。
  9. 我意识到过分的害怕和自我怀疑是毁掉我们才能的最大因素。
  10. 我们从跌倒中学会了走路，如果我们从不跌倒，也就永远学不会走路。
  11. 大部分人贫穷的主要原因就在于他们太担心失去。胜利者不害怕失败，但失败者害怕。失败是成功过程中的一个组成部分，如果避开失败，也就不会成功。
  12. 风险总是无处不在，要学会驾驭风险，而别总想回避风险。
  13. 富爸爸知道失败只会使他更强大，更精明。
  14. 得克萨斯人并不掩饰失败，而是愈挫愈奋，他们接受失败的现实并将失败转化为动力。失败激励得克萨斯人走向成功，而这个道理并不仅仅适用于得克萨斯人，它适用于所有的成功人士。
  15. 处理失败的不同方式造成了人们生活的差异，不仅是对金钱，对生活中的任何事情都是如此。
  16. 记住，轻松的道路往往会越走越艰难，而艰难的道路往往会越走越轻松。

# 2. 关于金钱

  1. 大多数人没有意识到，在生活中你挣了多少钱并不重要，重要的是你留下了多少钱。
  2. 大部分人在财务上不成功的最大原因是他们的做法过于安全。
  3. 今天，我们关注的是借钱来买我们想要的东西，而不是如何才能创造财富。

# 3. 关于工作

  1. 大多数人从不研究这个问题，他们去上班，挣工资，平衡收支，仅此而已，他们不明白自己为什么老缺钱，于是以为多挣点钱就能解决问题，但几乎没有人意识到缺乏财商教育才是问题的关键。
  2. 如果害怕没钱花，也先不要去找工作，挣几个小钱去消除恐惧，而要先问问自己：一份工作是最终消除这种恐惧的最佳解决办法吗。
  3. 以工资的多少来决定过什么样的生活不是真正的生活。认为工作会给你带来安全感其实是在欺骗自己。
  4. 工作只是面对长期问题的一种暂时的解决办法。
  5. 存在财务问题的人经常耗费一生为别人工作，其中许多人在他们不能工作时就变得一无所有。
  6. 从事你所学的专业的可怕后果在于，它会让你忘记关注自己的事业。人们耗尽一生去关注别人的事业并使他人致富。
  7. 大多数穷人和中产阶级财务保守（这意味着他们无法承担风险）的根本原因在于，他们没有经济基础。他们必须依附于工作，必须安全运作。
  8. 正如一句格言所说：“工作（job）一词就是‘比破产强一点’（justoverbroke）。
  9. 工人付出最大努力以免于被解雇，而雇主提供最低工资以防止工人辞职。

# 4. 关于负债

  1. 你必须明白资产和负债的区别，并且购买资产。如果你想致富，这一点你必须知道。这就是第一条规则，也是唯一一条规则。
  2. 富人获得资产，而穷人和中产阶级获得负债，只不过他们以为那些负债就是资产。
  3. 对成年人而言，把支出保持在低水平、减少借款并勤劳地工作会帮你打下一个稳固的资产基础。
  4. 富人与穷人一个重要的区别就是：富人最后才买奢侈品。
  5. 真正的奢侈品是对投资和积累真正资产的奖励。
  6. 他们没有意识到已有的某种思想或方法在昨天还是一种资产，今天却已经变成了负债。

# 5. 关于资产

  1. 你需要学会怎样才能使你的努力更直接地为你和你的家人带来好处。
  2. 富人买入资产。穷人只有支出。中产阶级购买自以为是资产的负债。
  3. 只有你把额外的收入用来购买可产生收入的资产，你才能获得真正的财务安全。
  4. 没有了职业的保障，他们也就失去了生活依靠。他们所认为的资产无法帮他们度过财务危机。
  5. 人实际的净资产要比他们自己认为的少得多。
  6. 一旦把1美元投入了资产项，就不要让它出来。
  7. 如果金钱不是真实的资产，那它又是什么？”“它是我们大家都认可的东西。”富爸爸会说。
  8. 实际上，你所拥有的唯一真正的资产就是你的头脑，这是我们能控制的最强有力的工具。
  9. 当你资金短缺时，让压力去发挥作用，而不要动用你的储蓄或资本。
  10. 如果你的资金很少而又想致富，你必须首先集中于一点，而不是追求平衡。
