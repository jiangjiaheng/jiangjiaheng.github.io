# 《前列腺疾病一本通 (著名专家常见病一本通系列)》

作者：赵忠宇

# 总述　拨开前列腺疾病的重重迷雾

# 总述-基本病理性

  1. 由于前列腺位于膀胱颈的下方、包绕着膀胱口与尿道结合部位，这部分因此被称为“尿道前列腺部”，即前列腺中间形成的管道构成尿道的上口部分。也可以这样说，前列腺扼守着尿道的上口，前列腺一旦有病，排尿首先受到影响，其原因就在于此。

  2. 因为人体的性活动如果过于频繁，就会使前列腺长期超负荷运转，造成前列腺身心疲惫、抵抗力下降，而病菌也就容易乘虚而入了。另外，由于前列腺借助射精管与人体的尿道相通，性活动过于频繁，射精管经常处于“闸门”开启的状态，尿道内的致病菌就可以逆行而上，侵入前列腺并生长繁殖，引起前列腺炎。

  3. 当前列腺受到炎症的侵扰后，首先就会导致精液质量下降、生育能力降低，甚至引起部分人无法生育。同时，慢性炎症还常引起人体生理功能的异常，比如阳痿、早泄等。专家提醒，要想将疾病拒之门外，保持生殖器官的局部卫生是非常重要也是非常必要的。此外，还应尽早矫正包茎、包皮过长等先天畸形，洁身自好以预防性传播疾病病菌的感染，只有做到了这些，前列腺才能远离疾病，保持健康。

  4. 腺体自身结构的改变：如果是体表发炎，很容易用肉眼或者医疗设备发现红肿热。但是前列腺在体内，发炎以后，不容易被看到，但实际上它会出现血管闭塞、腺体萎缩、纤维化。而这种病理改变造成的后果是药物不容易进去，细菌不容易根除。

# 总述-慢性非细菌性前列腺炎

  1. “慢性非细菌性前列腺炎”长期纵欲使前列腺反复充血所导致的一种前列腺炎症，最常见于18～25岁的未婚男性，由于这些男性长期大量频繁手淫，因此称为“手淫型前列腺炎”。

  2. 男科专家发现，有接近一半的慢性非细菌性前列腺炎病人会有焦虑、抑郁、恐惧、悲观等过度紧张的症状。有些同时伴有疼痛及神经衰弱的前列腺疾病人常常过于夸大躯体的不适合疼痛，以至于其自觉症状大于实际病情，这种情况被称为“紧张型前列腺炎”。专家指出，心理因素与年龄有关，年轻患者精神负担明显比年龄大的患者要重，这种精神上的“免疫力”往往直接影响到药物治疗的效果。

  3. 慢性前列腺炎病理改变比较复杂，许多因素都可以加重前列腺充血，如过度疲劳、饮食无节等，引起症状反复出现。但是，不管怎样反复，反复后的症状总是要比服药前轻，直至消失。由此，我们可以认为慢性前列腺炎其实不是不治之症，而是治疗不当就会“治不好”，不坚持治疗就容易“治不好”，治愈后不注意预防保健也会“治不好”。如果接受正规、科学的治疗，同时避免一些不良因素的影响，就很可能会彻底治好慢性前列腺炎。

  4. 男科专家指出，绝大多数慢性前列腺炎是非细菌性的，细菌性前列腺炎仅占到5％左右，且多为非特异性的普通细菌或机会性致病菌。

# 总述-不良习惯

  1. 无论是性生活过度还是长期禁欲，都会引发前列腺炎。前者是因为过度的性生活容易使前列腺过度充血，诱发前列腺炎；后者则是因为前列腺液长期滞留于前列腺内，给细菌的繁殖提供了良好的媒介和环境，从而诱发前列腺炎。

  2. 要避免酗酒和进食辛辣食物酒类和辣椒等辛辣食物能够引起前列腺血管的扩张、水肿或前列腺的抵抗力降低，进食这些食物后可引起前列腺的不适感，或有利于前列腺寄居菌群的大量生长繁殖，从而诱发急性前列腺炎或加重慢性前列腺炎。

  3. 不宜久坐或长时间骑车长期久坐或长时间骑车可造成对前列腺的直接压迫，导致前列腺的充血，从而诱发前列腺炎。

  4. 对此，专家提醒前列腺炎患者，尤其是长期处于紧张状态的男性，在接受治疗的同时，还需注意放松情绪，因为舒缓情绪也是一种疗法，而这种疗法对于前列腺炎的康复功不可没。

  5. 中医通常认为“久坐伤身”。而久坐是前列腺炎的直接诱因也已成为医学界的共识。久坐就容易缺乏运动，这将直接导致男性的气脉运行和血液流通受阻，进而造成男性阴部充血，引发前列腺充血、肿胀、发炎。特别是久坐沙发软椅，对男性生殖健康的危害更大。如今，越来越多的男性加入了“久坐族”的队伍。

  6. 除了“久坐”外，经常长时间站立也会导致男性前列腺长期充血，从而诱发前列腺炎。因此，当长期久坐、久站的男性发现自身有前列腺炎的症状时，应及时到医院专科诊疗，以免耽误病情；如果可以避免久坐、久站，就尽量避免。

  7. 长期憋尿，后患无穷。有的时候因为工作忙或是看到令人兴奋的体育比赛，经常会有了尿意也要坚持到最后，长期的憋尿，会造成膀胱过度充盈，使膀胱逼尿肌张力减弱，排尿发生困难，也容易诱发急性尿潴留。

  8. 此外，不恰当地使用抗生素会大量杀死体内的正常菌群，造成菌群的失调，使得一些可以致病的菌群或耐药菌群大量繁殖。所以，提醒前列腺疾病患者使用抗生素要谨慎，用药剂量不宜过大，过大会加重排尿困难。

  9. 另外，由于社会上的竞争激烈，而责任也会越来越大，从而导致心理压力增大，精神长期处于不安的境地，造成前列腺主动和被动充血，诱发前列腺炎。

  10. 研究发现，腹部肥胖会压迫前列腺，导致前列腺增生。现在许多人的饮食渐趋西方化，高蛋白、高脂肪食物摄入过多，容易对前列腺产生刺激。再加之中老年男性激素水平下降，脂肪代谢能力减弱，使脂肪堆积在腹部。

  11. 在前列腺炎的发生原因中，许多就与不良的生活习惯有关。久坐和长时间骑车就是其中的两个典型。从生理学的观点来看，座位可使血液循环变慢，尤其是会阴部的血液循环变慢，直接导致会阴及前列腺部慢性充血淤血。但一般时间的座位不会对身体有什么影响。如果因工作及其他原因，长期地较长时间保持座位，就会对前列腺造成一定的影响。这是因为会阴部、前列腺的充血，会使局部的代谢产物堆积，前列腺腺管阻塞，腺液排泄不畅，导致慢性前列腺炎的发生。

  12. 还有一个原因，有些男性全身的抵抗力比较低，这些人的特点，中医叫脾肾气虚，有这些虚证的一些病人容易感冒，往往旧的感冒没好，新的感冒又来。殊不知感冒往往和受凉密不可分，一旦受凉，交感神经兴奋，腺体的一些肌肉、血管收缩，导致前列腺液更加排泄不畅而加重前列腺的症状。另外，如盆底的肌肉有痔疮或有精索静脉曲张，以及一些精神因素都有可能导致前列腺疾病治疗困难。

# 总述-前列腺增生

  1. 前列腺增生的症状主要表现为两大类，一类是由于增生前列腺阻塞尿路产生的梗阻性症状；另一类是膀胱刺激症状。

  2. 前列腺增生的早期信号是尿频，尤其夜尿次数增多。一般来说，夜尿次数的多少往往与前列腺增生的程度成正比。原来不起夜的老人如果有夜间1～2次的排尿现象，往往表示早期梗阻来临。

  3. 前列腺增生早期会引起尿频、尿急等，是由于刺激以至压迫了后尿道和膀胱颈，如果能在此阶段治疗，效果最好；如没能及时治疗，则会随着病情的加重而出现各种并发症，梗阻的并发症主要有肾盂积水，感染，尿毒症等。

  4. 如此说来，可以用以下三句话概括前列腺增生与前列腺癌的关系：前列腺增生症不是癌症；前列腺增生症不会转化为癌症；前列腺增生症与前列腺癌有可能并存。

# 总述-前列腺炎的一些症状

  1. 尿路症状在尿道口常见有乳白色分泌物，尤其在排便等腹压增高的情况下出现。尿频，尿急，尿痛，尿有余沥，排尿时尿道有烧灼感，夜尿增多等。

  2. 精神抑郁症状出现精神不振、忧愁思虑、烦躁不安、失眠、多梦、健忘等，甚至焦虑、恐惧、愤怒、自卑，严重者有自杀倾向。可受医源性影响而症状加重或减轻。

  3. 其他症状主要有疲倦乏力、腰膝酸软、头晕耳鸣、大便秘或溏等症。

  4. 前列腺增生的主要症状有（1）排尿困难由于病情继续发展，每次排尿时间延长，射程较近，尿流逐渐变得细而无力。

  5. （2）尿频通常白天排尿6～7次，夜间2～3次或5～6次。这是因前列腺增生而引起尿道梗阻，每次排尿不能将膀胱内的尿完全排空，膀胱内经常有残余尿，有效容量减少，于是排尿间隔时间缩短，出现尿频。

# 第一章　重病要症，以防为主　——前列腺疾病人的预防保健方案

# 造成前列腺疾病的“六”硬伤

  1. ①嗜食辛辣食品

  2. ②吸烟无度

  3. 久坐

  4. 前列腺的位置决定了男人在很大程度上是“坐”在前列腺上的，所以经常久坐的男人前列腺负担较重。座位使血液循环变慢，导致前列腺慢性充血淤血，局部代谢产物堆积，前列腺腺管阻塞，腺液排泄不畅，导致慢性前列腺炎的发生。

  5. ④受凉前列腺有丰富的肾上腺受体，当它受凉时极易引起交感神经兴奋，导致肾上腺收缩，使尿道内压力增加，造成排尿困难，这又会对前列腺产生不良的影响。⑤长期便秘便秘者直肠内积聚大量粪便，会加重邻近的前列腺充血。同时，便秘者往往用力排便使腹压增加，压迫前列腺可使尿道变窄，排尿受阻，对前列腺健康不利。

  6. ⑥憋尿经常憋尿，可以使膀胱充盈胀大，压迫前列腺导致排尿无力，引起局部压力增大和血流不畅，加重前列腺肥大的症状。

# 保持前列腺健康的“六”护理

  1. ①过有规律的性生活性生活频繁的男性，发生急性前列腺炎的比率高达89.7％。相反，节制婚姻生活，致前列腺液大量“囤积”，时间长了导致前列腺过度扩张与充血，也可引发炎症。此外，体外排精、婚姻生活中断等，同样可使前列腺充血肿胀而引起炎症。

  2. ②多吃西红柿

  3. 在一项哈佛医学院针对47万个男人所做的研究中，研究人员发现吃烹调过的西红柿产品最多的人得前列腺癌的比例最低。

  4. ③多吃豆类食品日本男人死于前列腺癌的比率比美国男人低5倍。一般认为是由于他们的饮食多是低脂肪、高纤维食品。豆类食品在日本食物中十分普遍，豆腐、黄豆粉、豆浆，都是能有效遏止前列腺癌的食品。

  5. ④到医院做筛检专家建议广大男性朋友，在50岁之后应该每年到医院做两项检查：直肠检查和前列腺抗原的检查。如果本身有前列腺癌的家族史，最好从40岁就开始每年做这两项检查。

  6. ⑤小心荷尔蒙补品有许多宣传说睾丸激素治疗可以提升对婚姻生活的渴望，很多男性便冒险一试。然而高剂量的睾丸激素常常跟前列腺肥大和肿瘤有关，这一点容易被忽略。如果已经被前列腺问题所困扰，那么请远离这些荷尔蒙补品。

  7. ⑥阳光下运动研究表明，生活的过度压力会增加前列腺患病的机会，而当生活压力减缓时，前列腺症状就会得到缓解。所以建议男性朋友尽量保持放松的状态。户外阳光下的运动是身心放松的很好的方式，另外，适度地进行体育活动也有助于增强机体抵抗力，并可以改善前列腺局部的血液循环，保持前列腺的健康。

# 层层预防前列腺炎

  1. 若出现尿急、尿频、尿痛、直肠或会阴部疼痛症状，并多有恶寒发热的感觉，这就是急性前列腺炎，是非特异性细菌感染所致的急性炎症。在中医里属于热淋范畴。

  2. 有持续性的尿频、尿急、排尿困难和下腹部不适，或真性前列腺痛等症状，常规检查中无脓细胞，也无明显感染病理改变，这是前列腺痛和前列腺充血，属非细菌性前列腺炎的一种。

# 预防前列腺炎十注意

  1. （1）忌食辛辣食物、忌酒。

  2. （2）生活要有规律，起居有常，防止过度疲劳及感冒。

  3. （3）平时适当多饮水、多排尿，保持大小便通畅，以帮助前列腺分泌物排出。

  4. （4）不宜长时间骑摩托车或久坐，驾驶机动车时尤其注意坐椅的散热，避免温度过高，定时下车稍作活动及饮水排尿。

  5. （5）适当的性生活，避免房事过度。

  6. （6）坚持适当的锻炼，改善血液循环，利于局部炎症。

  7. （7）饮食合理，增加营养，增强免疫功能，防止复发。

  8. 每天最好能做2～3次提肛活动，即通过憋气自行努力把肛门提缩上去，每次5～10分钟，这是预防前列腺炎的一种既有效又简单的方法。

  9. （9）若发现兼有其他泌尿生殖系统炎症，应同时治疗，去除复发诱因。（10）由于前列腺炎而引起的不育症者，不宜温水坐浴。

# 日常预防

  1. 有研究表明，一次性冲动在性高潮期过后，盆腔充血消退的时间为15～30分钟；如果没有性高潮期，则盆腔充血的时间需半天到一天才能消退。这说明性冲动时，禁房事比不禁房事前列腺充血的时间要长得多。此外，精液中的前列腺液约占1/3，在房事排精的过程中，腺液中的部分炎性分泌物随之排出，这将有益于炎症的减轻和康复。如果长时间没有性生活，性兴奋常常受到抑制，有可能引起性功能的减退，甚至发生阳痿。所以前列腺炎患者严禁房事并不妥当。

  2. 多多喝水：前列腺炎患者应多补充水分。每天喝2～3升的矿泉水，以刺激尿流。如此可预防尿液潴留。

  3. 另一种水疗法是用温及冷水喷下腹及骨盆区域，以3分钟热，1分钟冷的方式交替喷淋。

  4. 在治疗上，无菌性前列腺炎应该说比较困难，通常医生建议采用综合疗法。首先，患者应消除不必要的思想顾虑，增强战胜疾病的信心。前列腺溢液者大多有神经衰弱症状，精神比较紧张，帮助这类病人弄清尿道口流白是排出的前列腺液，不是漏精，对纠正他们的神经衰弱非常重要。其次，患者应该建立起健康而有规律的生活秩序，防止过分的性欲思虑、过频的性生活及不正常的性行为。第三，积极参加体育锻炼，禁烟酒，忌辛辣。

  5. 另外，前列腺按摩也是治疗无菌性前列腺炎的重要手段，尤其对未婚或已婚却两地分居的前列腺炎患者，定期按摩前列腺以疏导淤积的前列腺液，这对于消除局部的炎症，减轻症状是十分有利的，按摩频率可维持在每周1次，连续按摩4～8周，可以达到一定的预期效果。

# 最好别让前列腺里长“石子”

  1. 当前列腺结石合并慢性前列腺炎时，可表现为会阴部、下腰部、阴囊、阴茎或肛门等部位的疼痛，排便时肛门痛加重，尿道外口有分泌物，会阴部受压时疼痛加重，伴有后尿道炎时可有尿频、尿急和尿痛，有时有血尿，尤其是终末血尿。形成脓肿时可有发热、尿潴留和会阴直肠部疼痛等症状。前列腺结石合并前列腺增生或尿道狭窄时，可表现为尿频、排尿困难、尿线细而无力等症状。

# “好吃懒做”是前列腺癌的主因

  1. 为减少前列腺癌发生的可能性，男科专家建议应该多食水果和新鲜蔬菜，增加纤维的摄入而减少糖的摄入，以鱼作为蛋白来源来减少动物脂肪的摄入，另外，应多参加有氧运动，尽量不要肥胖。

# 前列腺疾病引起急性尿潴留可不是闹着玩的

  1. 据有关文献报道约有50％～60％的前列腺增生症患者会发生急性尿潴留，发生时机多在感冒、劳累、饮酒、憋尿、房事或吃辛辣刺激食物之后。病人的症状通常表现为小腹胀痛，不能排尿等。

  2. 因憋尿、饮酒、疲劳等原因，患前列腺增生的病人的膀胱过度充盈，导致逼尿肌张力减弱，虽然有急切的尿意，但总是排不出来，这在医学上就叫做急性尿潴留。急性尿潴留大多数发生在半夜或者凌晨，患者起夜时发现不能自行排尿。

# 巩固保养

1。 不过，即使治疗效果再好，没有有效的巩固也会前功尽弃。对此，男科专家指出，对于康复的病人有两个要求：一是在疾病康复后的一个月，患者必须到医院复查及进行适当的前列腺保养。保养不但能帮助患者定期排放出前列腺液，还能从生活上细节上对患者进行一定的建议，有利于其养成良好的生活习惯。

# 第二章　运动健身，提升体能　——前列腺疾病人的运动保健方案

# 温和的运动最能保护前列腺

  1. 现在，医学专家们提出了用温和运动健身的新观点，原因是这种锻炼不仅能够促使大脑分泌更多的心理“愉快素”，给人们带来快乐，还能够逐渐增加体内血红蛋白细胞数量，改善大脑和心肺的生理功能，提高机体的免疫力，另外还可以避免年老体弱者在剧烈运动中的肢体扭伤、闪腰或骨折等运动损伤。同时也有研究证实，温和运动还可以保护前列腺。原因是温和运动改善了男性体内血液循环，使前列腺液分泌更旺盛，有助于前列腺的炎症消退。而长时间的、剧烈的、过度的运动，使儿茶酚胺和促肾上腺皮质激素分泌量增多，会造成老年男性前列腺充血、水肿，也会因过分耗能而“磨损”前列腺。

  2. 当然，温和运动的运动形式也是多样化的，既可全身整体运动，也可局部运动，如散步、快走、慢跑、爬山、跳绳、健身操、游泳、扭秧歌、跳交谊舞、保龄球、打羽毛球等，参与者可根据各自的体质和喜好来进行选择，最终目的无非是强身健体，“养护”前列腺。

  3. 而慢性前列腺炎如果久治不愈，还会并发精囊炎、阳痿和不育症等疾病，更是患者的一大心病。由于用抗生素治疗慢性前列腺炎的效果不十分理想，因此临床上常用一些物理方法如做医疗健身操、热敷等对该病进行辅助性治疗。

  4. 治疗慢性前列腺炎也许没有什么特效的药物或方法，但通过正规的治疗，积极的保健，会很好的提高广大男性朋友的生活质量。

# 久坐时适当起来动一动

  1. 一般时间的座位不会对身体有什么影响，但长期的久坐，就会对前列腺有一定影响。这是因为会阴部前列腺的充血，使得局部的代谢产物堆积，造成前列腺腺管阻塞，因此腺液排泄不畅，导致慢性前列腺炎的发生。

  2. 作为男性独有的重要器官的前列腺，既起到正常排泄腺液的重要作用，又有内分泌功能，还有性液排放的功能。

  3. 性生活不正常、长时间久坐、骑车压迫等，均可造成前列腺充血和发炎。

  4. 医学专家建议，坐四五十分钟左右就要起身活动一下，而且久坐的人要养成经常排尿的习惯，因为经常憋尿会引起尿路感染，也容易使前列腺受到压迫。像司机和摄影师等职业的男性，在憋了一段时间的尿之后，要尽快将膀胱排空，再补充一定量的水分，可以在某种程度上减少膀胱内细菌的增生。

  5. 调查发现，从事IT行业的人患病比例最高。原因是他们长时间使用电脑，一天坐上8个小时甚至更久是常事，久坐加上缺乏体育运动，使白领人士的气脉运行和血液流通受阻，容易造成阴部充血，引发一系列的问题，比如前列腺充血、肿胀、发炎等。

  6. 特别是久坐沙发软椅，对男性生殖健康的危害更大。坐在沙发软椅上，臀部深陷沙发中，填充物会包裹、压迫阴囊，静脉回流不畅，使得男士的整个生殖系统血液微循环受阻、新陈代谢减慢，导致新陈代谢产生的各种有害物质排泄不畅，淤积于前列腺之中，进而导致无菌性前列腺的发生和各种细菌性前列腺炎的加重。

  7. 所以，要想保护好前列腺，那就坚持适度的锻炼吧，特别是前列腺“最喜欢”的跑步运动。男科专家认为，跑步对于前列腺来说是一种很好的“按摩”方法，特别是在秋冬两季，如果能够适度地进行一些慢跑和弹跳的运动，可以在很大程度上减少前列腺疾病的发生。

# 经常练太极拳对保护前列腺大有裨益

  1. 打太极拳强调放松，放松成为这项运动的第一要义。另外，生理学和心理学等现代科学的一些研究也表明：当人处于完全放松状态时，大脑皮层的唤醒水平会下降，交感神经系统的有关功能也会下降，而副交感神经系统有关功则会上升，同时血气饱和度增加，血管舒张，血管容积增大，血管通透性明显改善，末梢血和流量增加，微循环得到改善。副交感神经功能一旦加强，胃肠蠕动分泌就会增强，胆汁分泌也就相应增多。打太极拳时的放松不是单纯某个部位的放松，而是要全身放松：无论是皮肤、肌肉、肌腱、筋膜、关节，还是内脏和全身其他器官、组织都要完全放松。只有做到身心的放松，才能舒缓心理的压力，使紧绷的神经得到松弛、令情志得到调畅、把郁结充分解除，这样一来，身体上的不适症状也会随之减轻甚至消失。太极拳站桩时，在基本的意守丹田基础之上，再以意导气经会阴穴至尾闾穴，沿督脉上冲至百会穴，内气沿副交感神经路线运行，内气循环不息地刺激副交感神经，并从丹田经会阴至尾闾，不断地按摩前列腺。如此一来，将体内的正气增强，也就相应的使抗病能力和自我修复能力得以提高。

# 前列腺收腹提肛保健操

  1. 在联系收腹提肛操时，患者先随着自己的呼吸作肛门的收缩和松弛练习，注意吸气的时候收

  2. 小腹缩肛门，呼气的时候放松小腹和肛门，这个动作连续做100次左右，早晚各做一次，以座位或卧位练习为好，需要持之以恒；其次是按摩小腹，做这个动作要两手叠按，右手在上，左掌心放在小腹正中，顺时针方向按摩100次，早晚各做一次；第三个动作是斜擦腹股沟，需要两手五指并拢，两手小指侧抵于髂骨骨棘，沿腹股沟向前下方斜擦100次，以小腹部有温热感为宜，每天可做一次；第四个动作是揉按曲骨穴，揉按时用右手中指峰在会阴毛际耻骨边的曲骨穴处用力按摩100次，以阴部有酸胀感为宜，该动作也可每天做一次；第五个动作是揉按肛门与阴囊之间的一个穴位，即会阴穴，会阴穴通称下丹田，用右手中指峰放在会阴处，轻轻揉动100次，使会阴部有微热发胀的感觉为宜，每天做一次即可。

  3. 上面这套收腹提肛操可以每半年或者一年练习一次。每次持续20～30天，每天练习的时间控制在1小时之内，也可以视个体差异一周间隔休息1～3天。切忌长期练习。

# 第三章　膳食调养，均衡搭配　——前列腺疾病人的膳食保健方案

# 前列腺炎与锌元素的“亲密”关系

  1. 食物中的海产品、瘦肉、粗粮、豆类植物、白瓜子、花生仁、南瓜子、芝麻等都含有丰富的锌，可以在家中准备这些小食品，闲暇时吃点零食。

  2. 苹果是我们都熟悉的一种水果，也是公认的营养丰富的水果，更是营养学家们向大众推荐的健康食物。苹果有一项非常重要的作用，那就是长期食用可治疗男性的慢性前列腺炎。

  3. 对于慢性前列腺炎患者来说，每天吃2～3个苹果，就可获得比较充足的锌元素，达到协同治疗前列腺炎、防止复发的目的，饮用苹果汁或食用果酱都可以。另外，其他含锌高的食物，如瘦肉、鸡蛋、花生米、核桃仁、芝麻、松子、葵花子等，可根据个人口味选择。

# 多吃番茄好处多

  1. 番茄里含有丰富的番茄红素，可以清除前列腺中的自由基，保护前列腺组织。因此，多吃番茄的男人等于给自己添加了前列腺“保护网”。一个成年人每天食用100～200克番茄，就能满足身体对番茄红素的需要。番茄红素需要经过加热和油脂烹调后，才更易于人体充分吸收，因此吃熟番茄才能起到作用。

  2. 据调查，在一些地中海国家，男人们前列腺发病率之所以很低，且性能力多半维持到60岁以上，主要就是因为当地盛产番茄，而男人们又喜欢吃的缘故。不过西红柿必须煮熟，并加油烹饪，这样番茄红素分子才能为身体利用。

# 对付前列腺，葱蒜立大功

  1. 研究表明，前列腺增生患者可以常常针对性地食用葱蒜类，以及南瓜子、番茄（该研究也指出，意大利面中常作为调味剂的西红柿也是许多人远离前列腺困扰的一个重要角色）、冬瓜子、红小豆等对前列腺保养有较好作用的食物。此外，此疾病虽然病在前列腺，但是很有可能还伴随着其他系统功能的下降，因此要尽可能使自己的食谱广泛，不要挑食、嗜食，这样才能对体内的各个系统会产生均衡的作用力，整体上进入良性循环。

# 大豆中的“神秘激素”可抑制前列腺癌

  1. 因此，专家建议，喜吃肉不必减少肉的摄入量，只要每日增加1杯豆浆就可对前列腺癌起到一定的预防作用。

# “肥”鱼可降低患前列腺癌的几率

  1. 研究人员发现这些鱼体内所含的一种被称为Ω3的脂肪酸对抑制前列腺癌很有作用。对此，研究人员沃克教授向路透社记者说：“我们的研究结论很简单，就是吃高脂肪鱼类。”

  2. 科学家指出，含有丰富Ω3脂肪酸的食物对健康很有好处。有些鱼类的体内带有Ω3脂肪酸。有证据显示，Ω3脂肪酸能够减少罹患心脏病的几率，还可能减少患肾脏癌的危险。研究者发现，常见于含高脂肪鱼类如鲑鱼、鲭鱼、金枪鱼等冷水鱼都是所谓的肥鱼、也就是脂肪多的鱼。这些鱼可能带有汞污染，但是大多数科学家说，对大部分人来讲，这些鱼的益处要超过它们的危险性。

# 每天一把南瓜子，可远离前列腺疾病

  1. 每日嚼食去壳生南瓜子40～50克，早、中、晚各一次，每次约15克。一周为一个疗程，可连续服2～3个疗程，无不良反应。服后可使尿急、尿频、尿痛及尿失禁等症状减轻，夜尿减少。

# 西瓜水，绿豆汤

  1. 西瓜水组成：西瓜1只。用法：西瓜洗净，剖开，以瓜代食。功效：清热利湿。主治：前列腺增生，属积热型，小便短少不畅，口干，发热者。

  2. 绿豆汤组成：绿豆100克。用法：绿豆洗净，置锅中，加清水500毫升，急火煮开10分钟，每次饮用10毫升，再加开水，代茶冲饮。功效：清热利湿，利小便。

# 第四章　顺应自然，冷暖相宜　——前列腺疾病人的四季保健方案

# 春天是前列腺疾病的头号健康话题

  1. 前列腺炎造成的症状很多，主要分为以下这四种：（1）尿路症状：以尿频、尿不尽、尿滴沥、尿痛、尿道灼热、尿急、排尿困难、尿黄为多见。（2）疼痛或不适症状：后尿道、会阴和肛门处坠胀不适感，下蹲、大便及长时间坐在椅凳上胀痛加重。慢性前列腺炎的疼痛感自尿道和会阴向其附近放射，以下腰疼痛最为多见。另外，阴茎、精索、睾丸、阴囊、小腹、腹股沟区、大腿、直肠等处均可受累。（3）性与生殖症状：晨起或大便时尿道口流出少许稀薄、乳白色、水样或黏稠分泌物，或伴有遗精、早泄、血精、射精障碍、性欲减退。（4）精神症状：易疲劳乏力、精神抑郁、记忆力减退、自信心下降等。

  2. 前列腺炎似乎越来越“青睐”年轻男性，有关数据表明，在年轻人当中约有25～30％会得这种病，为什么大量的年轻人会发病呢？业内专家大致总结了如下四方面的原因：①年轻人前列腺分泌旺盛，其中一部分没有结婚的人，没有正常的性生活，前列腺液淤积在内部，没有及时排出来。由于前列腺液是一种高营养物质，细菌一旦进入就会在里面繁殖裂变导致病发。②性生活频繁也容易引起前列腺充血，长期充血会使细胞的代谢受到影响，引起细胞对细菌的抵抗力下降，还会让细菌乘虚而入，所谓“蜜月病”说的就是这种情况。③现代生活节奏加快，尤其年轻人生活不规律，有一些不良的生活习惯如熬夜、吸烟、喝酒、吃辛辣食物、在电脑前久坐等，造成极度疲劳，这也会引起前列腺长期充血。④性生活不洁会引起性传染疾病，如果不及时治疗或症状不明显没有引起重视，也会转变成前列腺疾病。中老年人多为前列腺增生，有时合并为前列腺炎，比如排尿不畅容易引起细菌感染，引发前列腺炎。

  3. 春季前列腺疾病高发期保健“三多二勿”（1）多喝水，有利于前列腺分泌物的排出，可保前列腺的安全。（2）多保暖，可使用温水洗澡，应尽量避免一切使人受寒冷的事物。（3）多运动，避免久坐，最好的运动方式是走路。不宜骑自行车。（4）勿憋尿，憋尿不仅对肾脏有害处，而且使膀胱充盈，易诱发急性尿潴留。（5）勿食用刺激性的食物，多吃水果和蔬菜。

# 夏季别让前列腺炎给“忽悠”了

  1. 同时，专家强调，慢性前列腺炎病因复杂，患者的表现也不一样。有的患者有明显的如尿频、尿急、尿痛、尿等待、尿不尽或是局部疼痛等临床症状，但各项化验指标却很正常。有些患者白细胞、卵磷脂等化验指标异常，却没有明显症状。所以对前列腺炎的治疗，也应因人而异。如果仅仅依靠化验指标就诊断疾病，而症状并没有消除，这样的治疗无疑是没有意义的。而有些患者，临床症状一减轻就中断治疗，不坚持足程治疗，结果不但造成旧疾复发，还人为地增加了治疗难度。对前列腺炎的治疗，只有症状完全消失，同时各项化验指标均正常，才能说明前列腺炎真的离你而去了。

  2. 心理抑郁或焦虑，严重的抑郁及焦虑情绪会导致前列腺液排出减少或不畅，加重病情，影响疗效。（5）治疗不规范，不规范治疗或长期滥用抗生素，以及忽视合理治疗都会引起前列腺炎反复发作，不易治愈。

  3. 那么，如何判断前列腺炎是否治愈呢？临床上前列腺炎治愈的标准有以下几点：自觉症状消失；直肠指诊前列腺正常或改善；定位分段尿检查正常；前列腺涂片染色正常，细菌培养阴性。

  4. 治疗中特别不能“三天打鱼两天晒网”，“见好就收”，因为药物剂量不足、疗程不够，治疗不彻底，炎症并没有完全消除，极易导致疾病的复发，给后期的治疗带来困难。

  5. 因此，业内专家提醒广大男性，特别是患有前列腺疾病的患者，要慎吹空调。专家介绍，前列腺是人体中比较敏感的器官，具有典型的喜热怕冷的特点，尤其是中老年男性，更要从小处着手，把好预防前列腺疾病的这一关。

  6. 适当保护前列腺。即使是炎热的夏天，使用空调时也要注意身体保暖，特别要注意保护前列腺，晚上睡觉时要添加衣被，尤其注意小腹部位不能受凉。

  7. 养成良好的习惯。要适度进行运动，早晚可适当增加户外活动量，以改善人体的血液循环，提高免疫力，防止前列腺炎的发生。同时，要改变不良的生活习惯，如忍小便、少喝水、嗜烟酒、久坐不起等，这些习惯都对前列腺有一定的负面作用。

  8. 由于生活条件好了，一些男性朋友把空调的温度开得过低，特别是喜欢叉开腿坐的人，任由空调的冷风吹，一些慢性前列腺炎因此复发。在此提醒，除了控制好空调温度、适时添加衣物之外，男性朋友的正确坐姿对于保护前列腺也至关重要。

# 秋季男性宜养“腺

  1. 男性在秋天可以多吃高蛋白和滋阴补肾的食物使前列腺得到保养；另外，男性在秋季更应注意多饮水，多排尿。所以如果有尿频、尿急、尿痛、排尿不畅等症状应尽快就医，以免使炎症转为慢性。

  2. 此外，定期热水坐浴、补充维生素、加强体育锻炼，这可增强免疫功能，提高抗病能力，但是不要运用高40度以上的热水来坐浴，否则不但不能消炎，还很容易导致精子损伤。

  3. 梨、西瓜等都是秋天的应季水果，大多数人都难以抵挡其美妙的诱惑，但是由于梨和西瓜都属于寒凉水果，不适合前列腺疾病患者食用。因为对于久患前列腺疾病的人，往往处于肾气不足，阳气亏虚的状态，若再食用这些寒凉之品，则阳气无法固护，前列腺疾病也就容易反复，难以痊愈。

# 管好前列腺，这个冬季轻松过

  1. 所以，每到天气寒冷的时候，前列腺疾病门诊患者都会剧增。多数患者的治疗效果往往随着天气的逐渐变冷而不尽如人意。

  2. （1）饮水——每天饮水至少2升。天气转冷之后，由于不像夏天那样有过多汗液蒸发，许多人的饮水量也就不如从前。而饮水量减少会使尿液浓缩、排尿次数减少，这样，对前列腺及其他脏器（肾脏、膀胱等）等都会产生不利影响。所以，寒冷的季节也应该多饮水。一般人每天饮2升以上的开水或茶水，即可以通过排尿达到冲洗尿道、促进前列腺分泌物排出，起到预防前列腺感染的目的。

  3. （2）排尿——想尿时不要“忍”。许多人尿急时，由于一些因素不想及时去排，而是坚持“忍一会”，这种做法是不可取的。因为憋尿会让膀胱过度充盈，压迫前列腺，对于前列腺疾病患者来说，容易造成尿液反流，给高位脏器（肾脏和输尿管）带来危害，甚至造成肾衰竭的严重后果。

  4. （3）癖好——戒掉烟酒和辛辣。吸烟、喝酒、食辛辣食品，均对前列腺和尿道有很强的刺激作用，不但可以引起短暂的会阴部不舒服，还可引起前列腺和膀胱颈的充血、水肿，造成前列腺抵抗力降低。

  5. 因此，在寒冷的冬季，前列腺疾病患者一定不要靠吃辛辣食品和喝大量的酒来取暖，同时尽量不要吸烟。戒掉这些“癖好”，才能对前列腺疾病的恢复更有利。

  6. 局部保持温暖可使前列腺和精道的腔内压力降低，平滑肌纤维松弛，减少出口的阻力，使前列腺能够引流通畅。

  7. 海产品、瘦肉、粗粮、豆类植物、白瓜子、花生仁、南瓜子、芝麻等含有大量的锌。抗氧化剂可以保持前列腺健康。男士们的餐桌上应该多些粗粮、坚果、植物油、新鲜蔬菜和水果，以补充各类抗氧化剂。

  8. 冬季，是很多泌尿系统疾病发作或复发的季节，慢性前列腺炎也不例外，有关统计数据表明，有63％的慢性前列腺炎患者在冬季症状明显加重，而天气变暖之后，患者的症状又可以减轻。

  9. 冬季，人体为了抵御寒冷的侵袭，常处于不同程度的应激状态。如果患者忽略了自身的保暖，其交感神经兴奋性会明显增强，一方面使前列腺和膀胱颈部平滑肌收缩，导致膀胱颈部及前列腺部尿道压力增加，前列腺内尿液反流，使尿液中的有害物质进入前列腺，引发前列腺炎。另一方面，交感神经兴奋性过高时，可导致骨盆肌肉痉挛和前列腺局部血管扩张，使患者症状加重。因此冬季要为前列腺“保暖”，保持会阴局部的干燥温暖，有利于缓解盆腔肌肉的紧张，减轻排尿阻力，使前列腺引流通畅，缓解肌肉痉挛导致的盆腔疼痛不适。

  10. 另外，许多前列腺炎患者由于排尿不适造成尴尬，刻意控制饮水。其实尿液对尿道充分的“冲洗”是防止尿道病原微生物致病的关键因素。

  11. 治疗的过程中，前列腺疾病患者要保持乐观开朗的态度，不必认为得了前列腺疾病是件很糟糕的事，更不要觉得难为情。调整好心态，并积极配合医生的治疗，相信自己能够跨过前列腺疾病这道坎。

  12. 美国哈佛医学院等单位通过对15000名男性、经过18年的积累资料的分析发现，患有前列腺癌的人血液中钙的含量都比较高，因此就造成了他们体内维生素D的水平普遍偏低。

  13. 专家建议，每天最好利用15分钟的时间让身体沐浴在阳光里。

  14. 著名医学杂志《癌症研究》曾发表文章称，美国癌症专家研究发现：经常晒太阳的男性比不经常晒太阳的男性患前列腺癌的几率低一半，甚至低65％。研究表明，晒太阳可以使身体产生一种维生素D活性成分。阳光中的紫外线可以诱导维生素D的产生，而维生素D能够促进前列腺细胞的正常生长，抑制前列腺癌细胞入侵和扩散，因此有助于降低前列腺癌的发病率。

  15. 而对大多数人而言，只需要进行十几分钟的阳光照射就可以生成大量的维生素D，而不必将皮肤晒得通红。

# 第五章　健康习惯，日常做起　——前列腺疾病人的日常生活保健

# 40岁前预警前列腺增生

  1. 注重日常生活保健，有些措施对减轻病情及推迟该病的发生仍有一定价值。比如：注意防寒，预防感冒和上呼吸道感染等；绝对忌酒；少食辛辣；不憋尿；不过劳；避免久坐；适量饮水；慎用药物；及时治疗；按摩小腹。平时如果能做到以上几点，前列腺将会取得一定的自我保健功效，使正值家庭、事业均有所成就的中年男性摆脱“腺”状。

  2. 很多人不知道前列腺疾病在冬季最易发病和复发，即使出现了前列腺症状，也可能以为只是小毛病而不加理会。这样一来，就极易使急性前列腺炎迁延为慢性前列腺炎，治疗起来难度就会大得多。因此，男性朋友一定要了解前列腺炎的典型症状，一旦发现排尿异常、疼痛坠胀、性功能障碍等不适症状，就要及时找男科医生寻求治疗。

  3. 一般而言，前列腺癌多发生在65岁以上的高龄男性中，高发年龄平均在60岁以上。早期的前列腺癌，可以说是完全没有症状的，常常等到发现时已出现转移，一般转移到骨骼而引起疼痛。业内专家提醒：中老年男性定期对前列腺进行检查很重要，尤其在发现自己身体状况有异常的时候。

# 改变不良生活习惯，预防前列腺疾病

  1. 很大一部分前列腺疾病患者有着不良生活习惯，比如纵欲过度、缺乏运动、喝水太少、烟草酒精的不良刺激等，这些成为前列腺疾病发病的直接诱因。

  2. 改变不良生活习惯，是预防前列腺疾病的第一要务。专家建议，男性朋友在日常生活中，务必注意多饮水多运动，保持局部清洁，不过度憋尿，过和谐、节制的性生活。同时，有针对性地坚持选服前列腺保健品对防治前列腺疾病也非常重要。

  3. 同时，很多“成功男士”恶劣的生活习惯使得临床治疗与养护不能同步，导致反复发作，这也是前列腺疾病治疗屡屡失败的关键原因之一。

  4. 久坐的人，一定要经常“改变体位”。有些人觉得改变体位就是站起来散散步，其实给前列腺减压，卧床比散步的效果更好，最好找张床，沙发也行，躺上一会儿，对缓解前列腺充血的状况更加有效。

  5. 这是因为卧床后解除了腹压压力对前列腺的压迫，更能促进前列腺的血液循环，减少充血，还可以促进盆腔静脉的循环，这样不仅能够减轻盆腔疾病对前列腺的损害，还能减少盆腔疾病的发病率。

  6. 人体具有强大的酒精处理能力，大脑数小时后就会慢慢地从酒醉状态恢复过来。然而，前列腺却是充血快而恢复慢，一般要经过3～5天的时间才能渐渐地恢复原样。因为前列腺渗出的组织液需要一个较长的回吸收过程，可以说前列腺是酒精的最大受害者。所以，患者须尽可能少饮酒，最好不饮酒。

  7. 可见，吸烟虽然没有直接影响前列腺，但是可以影响精子的数量和质量，以及阴茎的血液循环，如此一来，就有理由认为吸烟会影响前列腺液的质量以及腺体血液循环，从而成为导致前列腺疾病的潜在因素之一。

# 第六章　两性健康，重中之重　——前列腺疾病人的两性生活保健方案

# 血精是慢性前列腺炎症引起的吗

  1. 由于精囊与前列腺在解剖上是紧密的毗邻关系，这两个器官在发炎时常是一对难兄难弟，即精囊炎往往与慢性前列腺炎合并发生。

  2. 慢性前列腺炎是引起血精的常见原因，而且往往是与患者房事过频或动作粗暴等有直接的关系。为此，提醒前列腺炎患者要节制房事，以防止或减少血精的发生。虽然血精症治疗并不困难，但科学而合理的护理可以促进其更快地痊愈和康复。所以，血精发生后不要惊恐，不要背思想包袱，但要及早治疗，通常情况下，病程越短的患者治疗效果就越好，相反，病程越长其治疗效果就相应越差。

# 前列腺炎与性生活

  1. 在射精中枢的指挥下，输精管、射精管、精囊、前列腺等器官的平滑肌收缩，迫使精液进入尿道的过程即为射精过程。

  2. 一般来说，发生射精疼痛的原因有：性交过于频繁，某些疾病如前列腺炎、精囊炎、尿道嵴炎、尿道结石或肿瘤等。

  3. 早泄可以由精神因素所致，但多数还是由于生殖系统的感染所致，常见的生殖系统炎症有前列腺炎、精囊炎、附睾炎等，这些病变使性阈值降低，稍一刺激即出现早泄。

  4. 要想清除前列腺内的炎症、促进局部的血液循环，定期排空前列腺液是很必要的。男科专家建议，在性生活前后抖动洗涤性器官，性生活时戴避孕套，避免精液直接进入女方的生殖道内，对女方产生潜在的危害；与此同时，使用避孕套还有另外一个好处，那就是由于慢性前列腺炎患者的前列腺免疫功能十分低下，极容易合并感染，避孕套可以帮助男性避免在性生活中感染女性阴道内的某些致病菌或机会性致病微生物。

  5. 但是还有一种情况，和性欲下降恰恰相反，是性欲高涨。这在早期或轻度慢性前列腺炎患者身上比较常见。

  6. 专家的解释是：由于慢性前列腺炎早期性功能旺盛，控制性活动的神经，因过于疲劳地工作而功能衰竭，于是由兴奋状态转入抑制。而重度慢性前列腺炎，细菌毒素和脓性物质直接刺激前列腺内部的神经组织，降低了它们的兴奋性，就不能很好地进行性生活。

  7. 也有的患者是因为前列腺及邻近性器官长时间广泛重度充血，直接影响尿道的收缩能力，干扰了前列腺排泄管肌肉的收缩能力，减弱了射精能力和迟缓射精时间，有的因为看到自己排出的是血性精液，心理上会非常恐惧，性功能也就难免会受到影响。

  8. 前列腺疾病人一般会在早期表现出性欲亢进。前列腺疾病的早期，患者可能会表现出与年龄不相符合的性欲增强，或者一贯性欲平常，突然变得强烈起来，这是因为炎症的刺激会引起精阜充血，导致阴茎的经常性勃起；另一方面，疾病导致前列腺功能紊乱，反馈性地引起睾丸功能一时加强。

  9. 另外，房事过多也会引起前列腺疾病。一个性发育正常的男性，不管有无异常刺激，都不可避免地经常有性冲动发生。尽管由于各种原因未能排出精液，使性兴奋得以宣泄，但生殖系统一系列相应的生理反应照样发生，出现各相关腺体分泌多、血管扩张充血等变化，前列腺即是相关的腺体之一。性生活本身会使前列腺长时间处于充血状态，在一而再、再而三的性冲动之下，前列腺分泌液增多和局部充血得不到宣泄，久而久之就可能导致前列腺的增生肥大，所以过于频繁和劳累的性生活也非常容易诱发前列腺炎。

  10. 年龄在50岁以下的男性，前列腺疾病不严重，无排尿不畅，身体条件和性功能正常的话，可以正常进行性生活，不过房事后最好用热水坐浴20分钟，温热刺激有利于局部血液循环，促使因性生活而充血的前列腺尽快消退，防止发生压迫尿道的症状。

# 性心理因素对慢性前列腺炎的影响

  3. 有些男性因怕损伤身体，性生活过度抑制，也会长时间自动兴奋造成前列腺被动充血。

  4. 性心理机能的紊乱可造成前列腺主动或被动充血，而反复慢性充血也是产生慢性前列腺炎的最基本因素之一。

  2. 多数未婚男性的自慰频度并没有他们自己想象的那样“频繁”。所以，

  3. 绝大多数成年男性只要把握一定的自慰频度，是不必担心诱发前列腺炎的，而且适度的自慰还可以帮助清除前列腺液，缓解前列腺的淤积，对保护或恢复前列腺功能具有一定的积极意义。由此可以看出，自慰对于前列腺的影响直接取决于能否有正确对待的态度。也就是说，自慰对于前列腺所起到的作用，直接取决于自身对此种现象的把握程度。虽然说自慰可以成为慢性前列腺炎的诱发因素，但如果能正确对待此种现象，则会在一定程度上对前列腺起到保护的作用。

  4. 对于频繁自慰的男性，应该把精力多放在事业和学习上，同时加强体育锻炼，并参加有益集体的文娱活动，注意生活调节，避免穿着太紧太小衣裤，按时睡眠，起床，避免刺激性食物，偶尔自慰，对身体不会有多大影响，但过分手淫或过分追求刺激会使生殖器长期处于充血状态，就会诱发前列腺炎。所以，还是要努力要提高自我控制和自我约束力。

  5. 性生活时前列腺要大量分泌前列腺液，所以会出现充血、水肿、肌肉收缩，一般情况下，需24～48小时充血水肿才能消除。

  6. 男科专家指出，适度有规律性地宣泄性冲动，是预防前列腺疾病的一种好方法。而衡量性生活频度是否适当的客观标准是，第二天早上是否精神饱满、身心愉快。

# 第七章　理顺情绪，心理减压　——前列腺疾病人的心理保健方案

# 五成前列腺疾病由自身心理障碍导致

  1. “血压高，血脂高，唯独薪水不高；成绩不突出，表现不突出，唯独腰椎间盘突出；大会不发言，小会不发言，唯独前列腺发炎。”

  2. 专家解释说，其实，前列腺炎本身不是什么致命的疾病，但精神上的压力却让男人不堪负重，这个疾病摧毁着他们的信心和尊严。

  3. 人们往往被忧虑、惧怕、怯懦、嫉妒和憎恨等不良情绪困扰，前列腺疾病则和这些心理因素直接相关。北京曙光男科医院专家指出：一个怀着长期压抑情绪的人，他的血中免疫球蛋白的水平比正常人低，他患前列腺炎，甚至癌症的可能性就比正常人大得多。出现前列腺炎的病症，患者心理负担重，担心前列腺炎会引发不育症、性功能障碍等。

  4. 如果一个人出现了前列腺炎的相关症状，首先要审视一下自己是否有不良的生活习惯，如是否酗酒、劳累、久坐等，经过一两周的自我调整，很多患者的症状就会自动消失。

  5. 学会做到心理平衡对于任何一种病症都有利无害，前列腺疾病自然也不例外。在具体的平衡技巧方面，不妨想想这句古话：祸兮福之所倚，福兮祸之所伏。

# 慢性前列腺炎造成的不良心理因素都有哪些

  1. 慢性前列腺炎患者，尤其是久治不愈的患者往往具有人格特性的改变，表现为应对方式的明显异常，可能有失眠、健忘、甚至出现胡思乱想和悲观失望等消极的应对态度，这方面的困惑有时甚至超过疾病本身的痛苦，并为此四处求医。在难以达到治愈的情况下，则又认为病情严重了，更加重了思想负担，二者互为因果，形成恶性循环，难以自拔。

  2. 其实，慢性前列腺炎对人体的危害，很大程度上是心理上的损害，特别是青年患者。由于缺乏对此病的正确认识，整日忧心忡忡、胡思乱想，甚至造成神经衰弱，个别染有手淫等不良习惯的青年，在疾病与精神双重影响下，就难免导致性功能减低，出现遗精、早泄或阳痿等，而性功能减低又加重思想负担，以致形成恶性循环。因而奉劝患前列腺炎的青年朋友，重要的是要解除思想压力和不必要的顾虑，树立起战胜疾病的信心，在医生的指导下进行有计划的治疗。

# 性格内向易患慢性前列腺炎

  1. 调查发现，慢性前列腺炎患者以内向性格居多。内向的人思虑太多，压力太大，长期压抑造成身体内环境的改变，身体抵抗力下降，加上内分泌紊乱，前列腺激素水平失衡，就引起前列腺炎。

  2. 黏液质类型患者，由于性格内向，多愁善感，办事认真，对该病所出现的任何症状都极为重视，对症状描述也极为细致，常常过分关注病的变化，为此担忧、烦恼、焦虑、不安，因而对本病的恐惧心理也异常加强，从而造成严重的心理负担，无端地加重病情，形成恶性循环。很多医生在临床工作中也有这样的体会，外向性格患者多对本病无过多忧虑，他们不过多在意病的变化，甚至工作一忙就把它放置脑后，由于情绪放松，能积极配合医生，因此这些人经适当治疗即可很快痊愈或不治而愈。

  3. 不可忽视的是，在男性中存在一种被称之为“病前性格”的人群，这种人的典型性格往往很内向，具体表现为缺乏自信心、多疑、多顾虑、不愿意与人交往、处事自卑又胆怯、容易受到外界的影响和暗示，不能客观地评价自己和环境。

  4. 很多性格内向的男性当出现一点儿泌尿系统的症状时，就会盲目把自己和前列腺炎对号入座，过分恐慌、担心。实际上，许多慢性前列腺炎临床症状的波动与心理因素的异常有关，如过度焦虑、悲伤、恐惧等。这样一来，不良的心理因素很可能占有主导地位，从而使男性的盆腔肌肉发生不自主的收缩，因而造成对膀胱、尿道的影响，出现尿急、尿频，下腹会阴部疼痛等症状，还能刺激机体的自主神经系统，造成前列腺液分泌量的改变。因此，这些在内向性格的男性身上更容易存在的“病前性格”可以使健康的男性出现前列腺炎的相关症状。对这部分男性来说，往往越有症状越紧张，形成不良的心理循环，长此以往，就可能真的诱发前列腺炎。

  5. 现代男性对前列腺疾病的认识存在两个极端，一是治疗过度，有点症状就花不少钱治疗，对疾病非常重视，导致心理压力过大。还有一个极端是一旦疾病发生后，不愿意或者不好意思就医，导致病情发展。在治疗中，同样疾病的病人，不同的心理状态，对治疗效果的影响也不同。有些病人由于缺乏相关的知识，把本来并非是气质性的问题而看得过重，这很可能源于其自身的心理“暗示”。

  6. 业内人士指出，抑郁症能够引发身体任何一个部位的症状。疾病与症状是两个不同的概念。症状是患者自我表述的，但这些症状不一定跟他的生理机能和身体的情况相符。有些患者本来未必是“前列腺炎”，但确实存在一系列的“前列腺炎”症状，而这些“前列腺炎”症状可能是抑郁症中的泌尿生殖系统症状，如尿频、尿急、夜尿多、遗精频繁及腰酸等。

  7. 有些临床医生在进行诊断时，往往过于强调生物学与临床物理化学检查的结果，而忽视了心理诊断，对心理疾病与精神疾病关注甚少，从而造成临床漏诊、误诊。医生关注患者的心理精神，可以使患者以积极的心态对待疾病，从另一方面也可以使那些“自认为”得了前列腺疾病的患者多了解相关的知识，做到心中有数。

  8. 治疗这个病目前无特效药，需要有一个规范的综合治疗，坚持用药，保证2～3个月足够疗程。一些患者症状一好转就停了药，不久又发病了，这样反反复复，久治不愈。

  9. 一些患者在治疗期间，不去改变不良生活习惯，当然也就治不好。慢性前列腺炎会引发小腹、会阴部疼痛，甚至睾丸、腹股沟、腰部疼痛，患者所承受的心理压力非常大，出现失眠、焦虑、抑郁，甚至悲观失望等情绪，曾经有一位患者因为难以忍受尿道刺痛，两次要自杀。另外，不停地换医生也是成为慢性前列腺炎难治的原因之一。

  10. 其实，男科专家普遍认为这个病是可以治愈的，只要症状消除了，三次前列腺液复查基本正常，就算是已经完全康复了。

  11. 男科专家强调，判断慢性前列腺炎病情好转的指标，症状消除是最主要的，比如排尿的障碍，会阴部的隐痛，腰酸背痛，这些症状都没有了，只是白细胞有点高，可通过自身的免疫调整，可能会自然康复。

# 解除焦虑，慢性前列腺炎不药而愈

  1. 多数病人除了有会阴疼痛、坠胀不适等症状外，还常常伴有紧张、焦虑、抑郁等情绪问题。而这些不良情绪对于疾病的恢复是有百害而无一利的，相反，如果拥有好的情绪，则会对身体的康复起到不可估量的作用。

  2. 对付慢性前列腺炎，无论中医、西医，均无特效疗法。之所以缠绵难愈，除了解剖、细菌及生理等方面的原因之外，情感方面的原因，尤其是紧张、焦虑、抑郁等情绪，也增加了治疗的难度。这在临床上屡见不鲜。

  3. 类似宋波和平涛那样的慢性前列腺炎病人，由于工作、学习压力的加大，经常出现外生殖器、睾丸、会阴和腰部的种种不适，加上尿急、尿痛、尿频、尿道流白色液体等症状反复发作，甚至出现性功能障碍，使得他们烦躁不安、痛苦异常。

  4. 因此，米勒认为，紧张因素是慢性前列腺炎的重要病因。他不让病人服药和按摩前列腺，只采取消除紧张情绪的行为疗法给本组病人治病。经过几个月的随访，其中有80％的病人自诉症状减轻或痊愈。根据这一研究成果，他指出，在慢性前列腺炎的诸多病因（如病原微生物感染、充血、免疫等）中，紧张或许是更为重要的因素。

  5. 目前，不少学者赞同米勒的观点，认为心理因素在慢性非细菌性前列腺炎的病因学中起着重要作用，他们认为，把慢性前列腺炎叫做“紧张性前列腺炎”可能更为确切。

  6. 有各种慢性前列腺炎症状的人，如果学会进行自我心理调节，通过各种方式缓解紧张、焦虑、抑郁的情绪，那么，或许有一天会突然发现，那些曾经令你痛苦不堪、缠绵不愈的疼痛、坠胀不适、尿频、尿急不知何时已经离你而去了。

# 紧张会导致慢性前列腺炎吗

  1. 据男科专家介绍，前列腺组织中分布有自主神经，当自主神经系统受“紧张”刺激时，腺体分泌量增加，肌肉收缩增强，患者就会表现出不适与尿频，当不适感觉达到一定的程度时便产生疼痛感觉。实验数据表明紧张可能是慢性前列腺炎的病因，短时或长期的紧张刺激可以引起猫的前列腺出现类似于人类前列腺炎的反应。

  2. 前列腺炎多发于中年男性，这和中年人社会责任大，精神压力重有着直接的关系。

  3. 需要提醒的是，有些媒体或广告对该病的宣传缺乏科学性，或夸大疾病的危害。这些宣传，不是造成患者精神紧张，就是增加患者的侥幸心理，跟着广告求医、吃药，致使自己上当受骗，从而动摇了治疗信心，加重了心理负担。所以，患者有必要学习和了解此病，懂得应该怎样就医和遵医嘱坚持治疗，做到心中有数。

  4. 不少患者由于性格内向，害羞，不好意思把病情讲出来。患了前列腺疾病后自己忍着，容易产生焦虑、抑郁、恐怖、悲观等心理障碍。患者只有面对现实，及时求医，向大夫如实告知自己疾病的症状和相关病情，才能预防和扫除心理障碍，积极配合治疗。

  5. 有的患者因有不当性行为，不愿到正规医院求治；有的患者出现了性功能障碍，担心影响夫妻关系和生育，而终日心神不定，惴惴不安。这些心理障碍和情绪使患者的精神负担加重，直接影响治疗，而且会导致疗效越来越差。因此，这类慢性前列腺炎患者只有积极并彻底地纠正自己的不良性行为，重新树立健康向上的性观念和性意识，才能从精神上得到彻底解脱。

# 不良情绪对前列腺疾病百害而无一利

  1. 从科学的认知看，一些慢性前列腺疾病的患者一遇到焦虑、恐惧、愤怒或悲伤情绪时，就会出现症状明显加重的迹象；一旦心情愉快时，症状就又明显减轻，甚至消失。由此看来，这就是心理影响了生理。不良情绪使机体的自主神经紊乱，使前列腺分泌大大增加，同时，局部及周围肌肉组织发生不自主的收缩，刺激膀胱和尿道，从而产生尿频，尿急及肛门周围，会阴处的疼痛刺激症状，但做前列腺常规检查，并没有发现什么异常，这类前列腺炎症状往往就是身心疾病范畴了。

  2. 在本病的康复过程中，保持良好的心态，情绪稳定尤为重要。一些患慢性前列腺炎及前列腺增生的病人，常因本病病程长，短期治疗效果不明显，或易复发而失去信心，产生较重的思想负担及恐惧、忧虑、烦躁易怒等不良情绪。有的患者甚至认为自己患了不治之症，终日忧心忡忡，严重影响病人的正常生活和工作，对身心健康非常有害。

  3. 总之，不良情绪对前列腺疾病百害无益。因此，患病后千万不要害怕和过分担忧，要树立战胜疾病的信心，在战略上要藐视它，同时要按医生的要求去做，积极配合治疗，保持情绪稳定，注意饮食宜忌，节制性欲，适当地参加体育锻炼，提高机体正气，相信前列腺疾病是可以治愈的。

# 4个方法可防止被坏情绪传染

  1. 在此提供一些简单易行的方法，或许对于你避免受到负面情绪的传染起到一些作用。

  2. （1）转移你的注意力在遇到倒霉的事情时，人往往是越想越气愤，越想越难受，其实，这个时候就不妨把那件事情丢开，把注意力分散到别处去，比如去看看电视，唱唱歌，洗个澡等。当做了一些轻松的事情后，你就会渐渐地发现原来情况没有那么糟，生活还是挺美好的。

  3. （2）离开激怒的现场人一旦火气上来，很可能一个眼神、一点小动作、一句话都会成为“硝烟弥漫”的导火线。所以，三十六计，走为上策。等到让自己冷静之后，再仔细想一下，也许你会发觉没什么大不了的。就像美国第三任总统杰弗逊说的那样：“先数到10，然后再说话，假如怒火中烧，那就数到100。”当感到怒不可遏的时候，不妨试试，也许能让你绷紧的弦稍稍地松弛下来。

  4. （3）让怒气合理宣泄出去有了怒气，总是压抑着对身心都没有好处。如果你的怒气膨胀起来，不妨把自己单独关在一间房屋里或者是跑到没有人的空旷的地方，，大喊大叫几声，任意释放怒气。

  5. （4）找点新鲜给自己心理学研究发现，新鲜感会让我们心情转好，比如换换发型，买一两件新衣服，或者重新布置布置办公桌，或者给家里购置一些新奇的物品和摆设，即便这些东西并没有多少实用性，但这种新鲜感会让你的精神得到一定程度的满足。

# 第八章　诊断治疗，健康是金

# 治疗前列腺炎注意事项

  1. 切忌见好就收。慢性前列腺炎是一种比较复杂难治的疾病，一般需要治疗1～3个月，并且在治疗有效后还要继续巩固治疗一段时间。有些患者由于临床症状明显减轻或消失就拒绝继续治疗，实际上，患者的主观感觉与客观检查经常会不一致，一旦时机成熟，病情还会复发。
  2. 要养成良好的生活习惯。良好的生活习惯包括：保持充足的睡眠，适度参加体育锻炼，不大量食用刺激性食物，防止过度疲劳，性生活有规律，不酗酒，不久坐或长时间骑车，不久居寒冷潮湿的住所，预防流行性感冒等感染性疾病。
  3. 正确认识，消除心理负担。很多慢性前列腺炎患者由于疾病久治不愈往往产生严重的心理负担，50％左右的患者还会出现不同程度的精神障碍，这些对疾病的治疗与康复非常不利。因此，只有情绪得到放松，心理没有过重的负担，疾病的治愈才会更有希望。
  4. 中药治疗：中药治疗的关键是要抓住祛邪这一主要矛盾。治疗原则是利湿通淋，清热泻火、祛淤排浊、活血化瘀等，以直接根除病灶、消除炎症、改善体内循环等，达到治病求本的目的，急、慢性前列腺炎患者都可以采用。

# 排尿症状

  1. 排尿次数增多，医学名词称之为尿频。这是良性前列腺增生的早期症状之一，主要表现为起夜次数增多，常常超过两次以上。
  2. 专家介绍，排尿费力，常常需要增加腹压帮助膀胱排尿，在医学上称之为排尿困难。其原因是当增大的前列腺向后尿道或向膀胱出口的部位突出并压迫上述部位，使后尿道延长、变窄和扭曲，影响尿流顺利排出体外。表现为一系列的排尿梗阻现象，如排尿等待、尿线变细或分叉、尿液排出距离缩短、排尿时间延长、排尿费力、分段排尿，严重者甚至尿淋漓不尽，也可出现明显尿频及不自主排尿即尿失禁现象。
  3. 膀胱充满尿液却突然不能自主排尿，越是着急越是排不出尿来，医生称之为急性尿潴留。它常发生在良性前列腺增生患者受凉、饮酒、憋尿等刺激后，大量小便潴留膀胱内，会伴有小腹部胀痛。
  4. 发现尿中有带血的现象，也就是医学上所说的血尿。是由于前列腺组织增生，其表面血管扩张以及尿路感染，引起局部出血，而出现血尿。约有30％的良性前列腺增生患者的首发症状为血尿。
  5. 已经有很长时间排尿不通畅，也有可能是前列腺“肥胖”所致。尿液中的小结晶易在膀胱内停留，成为核心而形成膀胱结石，其发生率可达10％以上。俗话说“流水不腐、户枢不蠹”，长期排尿不通畅，尤其是膀胱内有大量残留尿液时，容易引起膀胱感染，患者出现尿频、尿急、尿痛甚至发热，同时排尿困难的症状加重。膀胱内长期残留小便以及排尿困难，造成尿路内压增高，可压迫肾脏组织，使其泌尿功能发生损害，最终将有可能导致尿毒症。

# 治疗误区

  1. 不过业内专家还是提醒患者，前列腺治疗无捷径可走，规范治疗是基本原则。另外，一定要纠正自己在前列腺疾病上的一些错误认识，在医生的指导下正确应对，既不要过度治疗；也不要在应该手术治疗时拖延不决，延误病情。特别是以下误区，一定要避免，这些都是造成“二手患者”的原因：
  2. 认为抗生素是“救命神仙”：有些患者总认为抗生素是消炎药，有了前列腺炎就得用它。其实前列腺炎中有90％左右的病人都是无菌性的，真正可以找到病原体的感染性炎症不足10％。无菌性前列腺炎用抗生素不但不起消炎作用，还让体内局部菌群失调，加重病情的难治程度。
  3. 相信一次性“根治”：绝大多数是非感染性炎症，外面通常有很厚的脂质包膜包裹，药物不易渗至小腺管内，所以治疗起来不可能速战速决，想一次就治好是不可能的。
  4. 治疗不系统：前列腺炎病因复杂，加之特殊的生理解剖特点，前列腺炎往往不可能用一种方法就能治好，如有菌性（感染性）前列腺炎，在用抗生素的同时，要改善局部血循环，增强机体免疫功能，提高吞噬细胞功能等。

# 注意事项

  1. 有始有终，足程治疗是前列腺疾病的治疗的关键所在，男科专家提醒广大患者，越是临近治疗康复期，患者越要加强巩固和保养，因为巩固和保养是给患者提供康复的“双保险”，只有这样，才能把炎症复发的可能性降至最低。
  2. 慢性前列腺炎症状表现不一，不同的前列腺炎患者的症状表现相差很大。其中，急性前列腺炎症状表现比较明显，而慢性前列腺炎症状表现呈现多样化、症状和炎症轻重程度也没有规律性可循。
  3. 虽然部分前列腺炎的确难以治愈，但是通过正确认识它的本质，配合科学的预防、治疗和生活保健措施，可以控制病情的发展，甚至达到逐步治愈的目的。但是前列腺疾病因复杂，特别是慢性前列腺炎症状表现不一，病因隐晦，确诊比较困难。因此，在种种克服前列腺疾病的难题面前，只有正规的专业医院才能担此重任，对该病予以规范的治疗，使患者身体恢复健康。
