      # 系统核心

      这个模块是整个操作系统的底层规则。

      它只回答三个问题：

      1. 我往哪里走
      2. 我怎么走
      3. 哪些边界不能突破

      # 核心立场

      # 1. 保守主义是方向

      保守主义不是停滞，而是优先信任被时间和现实反复验证过的结构。

      默认判断：

      • 稳定比刺激重要
      • 秩序比情绪重要
      • 长期比短期重要
      • 可持续比爆发重要

      # 2. 效率主义是方法

      效率不是压榨自己，而是保护有限资源。

      过去我习惯追求结果最优解：从结果上看接近 90 分，甚至看起来完美。

      但结果最优的解法，往往也是综合成本最高的解法。

      好的方法应该：

      • 成本更低
      • 更容易执行
      • 更容易复用
      • 更不依赖意志力

      所以效率主义的核心不是把每件事做到最好，而是在时间、精力和金钱的总成本里，找到真正划算的解法。

      # 3. 创作型人生是长期方向

      体验不是错误，但不能成为系统中心。

      长期方向是从体验型人生转向创作型人生：

      • 少一点消费和刺激
      • 多一点输出和构建
      • 少一点短期满足
      • 多一点长期积累

      # 判断顺序

      # 1. 人性优先

      人性不会因为理性更强就消失。

      趋利避害、损失厌恶、虚荣、懒惰、比较、短期诱惑，都是长期存在的底层结构。

      系统设计不能假装人性不存在。

      # 2. 经验优先

      现实里反复验证过的经验，比纯逻辑推演更可靠。

      推演可以辅助判断，但不能替代现实反馈。

      # 3. 理性负责调整

      理性的职责不是审判感受，也不是消灭人性。

      理性应该负责：

      • 分析局部问题
      • 权衡利弊
      • 调整路径
      • 设计规则
      • 降低系统风险

      # 运行规则

      # 1. 60分原则

      能稳定落地的 60 分，通常比高成本追求 90 分更有价值。

      默认规则：

      • 完成优先于完美
      • 够用之后停止优化
      • 只有额外成本很低时才继续提升
      • 理想化方案如果无法执行，实际价值就是零

      # 2. 成本最优原则

      解决问题时，优先追求成本最优解，而不是结果最优解。

      成本包括：

      • 时间成本
      • 精力成本
      • 金钱成本
      • 情绪成本
      • 后续维护成本

      有些方案从结果上看不够完美，但如果它低消耗、可执行、能解决主要问题，它就是更适合系统的方案。

      默认规则：

      • 先问成本，再看结果
      • 不为 90 分结果支付过高成本
      • 能用低成本解决的问题，不升级成高成本问题
      • 解决主要矛盾之后，允许结果不完美

      # 3. 连续性原则

      长期稳定来自持续，而不是爆发。

      系统风险不按平均值结算，而按最坏一天结算。

      默认规则：

      • 允许慢
      • 不允许失控
      • 不用一次性消耗换未来免维护
      • 当效率和连续性冲突时，选择连续性

      # 4. 区间原则

      现实里的确定性不是一个点，而是一个区间。

      涉及不确定系统时，先预设可接受范围：

      • 时间区间
      • 结果区间
      • 风险区间
      • 行动频率

      结果在区间内，不追加情绪判断；超出区间，再分析和干预。

      # 5. 分工原则

      个人不是闭环系统。

      不是所有事都应该自己扛。

      先判断：

      1. 这件事是否需要专业能力
      2. 是否必须由我亲自完成
      3. 是否可以交给协作、工具或系统

      拒绝亲力亲为幻觉。

      # 系统边界

      • 不为短期情绪高峰牺牲长期稳定
      • 不接受不可撤回、不可验证的高风险押注
      • 不用爆发式消耗换表面进步
      • 不透支身体和睡眠
      • 不在高情绪状态下做重大决定

      # 一句话总结

      系统核心是：方向保守，方法高效，节奏连续，边界清楚。
